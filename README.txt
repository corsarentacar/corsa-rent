
# Corsa Rent a Car Landing Page

Esta es una página de aterrizaje simple que redirige automáticamente al número de WhatsApp de Corsa Rent a Car.

## ¿Cómo usarlo?

1. Sube los archivos a un repositorio público en GitHub.
2. Ve a la configuración del repositorio > Pages.
3. Activa GitHub Pages desde la rama `main` o `master`.
4. Sube el logo como `Corsa-logo.png`.
5. Espera unos minutos y GitHub generará una URL pública.
6. Conecta tu dominio de GoDaddy a esa URL (GitHub Pages te da instrucciones en la misma sección).
7. Usa ese dominio en tus campañas de Google Ads.
